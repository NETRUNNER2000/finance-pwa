# 💰 Project Truesight – Personal Finance PWA

---

## 🧠 WHAT IS THIS?

**Project Truesight** is a **Next.js Progressive Web App (PWA)** designed to **help you track, visualize, and optimize your finances**.  

- Track expenses, income, and account balances  
- Categorize spending for better insights  
- Visual dashboards with charts for analysis  
- Offline support via PWA caching  
- Minimal, fast, and secure  

---

## ⚙️ TECHNOLOGY STACK

**Frontend**  
- Next.js / React  
- Tailwind CSS / Styled Components  
- TypeScript  
- Chart.js / Recharts  

**Backend**  
- Node.js / Express API  
- SQLite / PostgreSQL (configurable)  
- REST / JSON endpoints  

**Features**  
- PWA support (offline + installable)  
- Local storage + Sync  
- Responsive design (mobile-first)  
- Authentication & secure data handling  

---

## 🧩 FEATURES

### 💳 Expense & Income Tracker
- Add, edit, and delete transactions  
- Tag and categorize each entry  
- Filter by date, category, or account  

### 📊 Visual Insights
- Bar charts, pie charts, and trend lines  
- Automatic calculation of totals & balances  
- Highlights overspending patterns  

### ⚡ Offline & PWA
- Works offline with service worker caching  
- Installable on mobile & desktop  
- Fast, lightweight, minimal loading times  

### 🔐 Security
- Local data storage + optional cloud sync  
- Password-protected accounts  
- Simple, safe authentication
